EGE AI — MVP сайта

Как запустить:
1. Распакуй архив.
2. Открой файл index.html двойным нажатием.
3. Нажми "Начать бесплатно" или "Войти".

Что внутри:
- index.html — структура сайта
- style.css — дизайн
- script.js — простая логика
- README.txt — инструкция

Что такое MVP:
MVP — первая минимальная рабочая версия продукта. Она уже выглядит как сайт и позволяет показать идею людям, но это еще не полноценный сервис с настоящим ИИ, регистрацией и оплатой.

Что нужно добавить позже:
- настоящий AI через API
- регистрацию пользователей
- базу данных
- оплату подписки
- сохранение прогресса
